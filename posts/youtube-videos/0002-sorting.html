Statement: A tree with $n!$ leaves will have a depth of $\Omega(n\log{}n)$

Proof

A tree with height $h$ has a maximum of $2^h$ leaves. Thus
$n! \le 2^h$
which means
$h \gt log (n!)$
i.e. $h = \Omega(log(n!))$
Now, by Stirling's approximation,
$n! ~ \sqrt{2\pi n}(\frac{n}{e})^n$
So,
$log(n!) = \theta(\log(\sqrt{2\pi n}(\frac{n}{e})^n))$
$= \theta (log{\sqrt{2\pi n}} + n \log\frac{n}{e})$
$= \theta (log\sqrt n + n \log n)$
$= \theta ( n \log n)$
Thus, $h = \Omega(n \log n)$
